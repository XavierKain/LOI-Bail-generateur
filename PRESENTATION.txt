🎯 OBJECTIF
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

Automatiser la génération de documents LOI à partir de fichiers Excel sources.

✨ FONCTIONNALITÉS
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  ✓ Interface web intuitive (Streamlit)
  ✓ Extraction automatique des données Excel
  ✓ Calculs automatiques (paliers, dates, surfaces, type bail)
  ✓ Remplacement intelligent des placeholders
  ✓ Gestion des sections optionnelles (suppression auto)
  ✓ Marquage rouge des données manquantes
  ✓ Headers/Footers dynamiques (10 sociétés préconfigurées)
  ✓ Téléchargement direct du document DOCX

🚀 DÉMARRAGE RAPIDE
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  1. Installation des dépendances:
     $ pip install -r requirements.txt

  2. Lancement de l'application:
     $ ./run.sh
     ou
     $ streamlit run app.py

  3. Ouvrir le navigateur:
     → http://localhost:8501

  4. Upload → Vérifier → Générer → Télécharger

📊 VARIABLES EXTRAITES
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

  • Informations preneur:    Nom, Société, Enseigne
  • Informations bail:        Durée, Type, Dates
  • Surfaces:                  Totale, RDC, R-1
  • Loyers:                    Base + 6 années
  • Paliers:                   Calculés automatiquement (1-6)
  • Conditions:                Suspensives (1-4)
  • Autres:                    Franchise, Travaux, Paiement, etc.

  → Total: 40 variables

🎨 SYSTÈME DE COULEURS

               • Avec données    → Affichée en noir
               • Sans données    → Supprimée

               • Avec données    → Remplacée
               • Sans données    → Marquée en rouge

  🔴 ROUGE   = Données manquantes
               → À compléter manuellement
